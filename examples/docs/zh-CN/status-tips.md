## StatusTips 状态提示

### 基础用法

:::demo 

```html
<yl-status-tips status="warning">
    待审核
</yl-status-tips> 

```

:::

### Attributes

| 参数   | 说明                         | 类型   | 可选值             | 默认值             |
| ------ | ---------------------------- | ------ | ------------- |----------- |
| status | 状态 | String |['success', 'warning', 'danger', 'info', 'gray']|'success'|
| width | 宽 | String |---|'80px'|
| height | 高 | String |---|'35px'|
| top | top | String |---|'10px'|
| right | right | String |---|'10px'|
